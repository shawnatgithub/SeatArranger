# 模块4_营销与成交

> 使命：让目标用户看到产品、愿意上手、并完成付费或关键转化。
> 通关标准（对应关卡3/4）：完成上架素材 + 通过陌生人冷启动测试 + 获得首批种子用户。
> 输入：来自模块2的"差异化定位"和模块3的"可用MVP"
> 本产品范例：座位排布应用（会议 / 宴请）

## 一、核心信条
- 营销不是上架后才开始，从模块1验证需求时就该边做边攒粉（公开构建）。
- 一句话说不清产品价值，就还没准备好营销——先回去打磨定位。
- 冷启动靠"精准小池塘"，不靠"大海捞针"。

## 二、上架/发布素材清单（核心交付物，通关关卡3）
- [ ] 应用图标（清晰、一眼看懂是排座工具）
- [ ] 名称 + 一句话副标题（含核心差异点，如"拖拽排座·改座不重排"）
- [ ] 应用截图 3-5 张（突出"拖拽布置"这个爽点，配简短说明文字）
- [ ] 详情页文案：痛点开头 → 核心功能 → 适用场景（会议/宴请）→ 行动号召
- [ ] 一段30秒演示视频/GIF（拖拽排座的动态最有说服力）

## 三、一句话价值主张（营销的地基）
> 模板：帮【谁】在【什么场景】下【更快/更好地完成什么】。
- **范例**：帮活动执行和宴会策划，用拖拽几分钟搞定会议和宴请的座位排布，改座不用重排。

## 四、冷启动渠道（找你的"精准小池塘"）
| 渠道      | 适合人群            | 怎么做                              | 优先级 |
| --------- | ------------------- | ----------------------------------- | ------ |
| 小红书    | 婚庆/宴会策划、新人 | 发"排座技巧/避坑"内容，自然带出工具 | 高     |
| 知乎/贴吧 | 行政、会务          | 回答"如何快速排座位"类问题          | 中     |
| 微信社群  | 婚庆同行群、行政群  | 进群提供价值后再介绍                | 高     |
| 公开构建  | 独立开发者/早期用户 | 在社交平台连载开发进度，攒早鸟      | 中     |

- [ ] 准备一句邀请话术：____

## 六、转化路径设计（从看到到付费）
看到内容 → 点进详情页 → 下载/打开 → 完成第一次成功排座（关键"啊哈时刻"）→ 留存 → 付费
- **关键**：确保用户最短路径就能体验到"拖拽排座成功"那一下爽感，这是转化命门。

## 七、定价落地（承接模块2）

- 付费点设计：____（如基础免费，多桌联动/AI智能排座/导出高清图为付费）

## 八、冷启动测试（通关关卡3的硬指标）
